# Archimedes-1 

Description:

Archimedes-1 is our second large scale project after last years can-sat entry. Archimedes-1 is a fully custom designed remotley controlled submarine. Built around an ESP32 microntroller as the centeral brain, all the electronics will be housed inside a sealed accrylic tube with O-ring end caps to ensure that everything stays watertight. Archimedes-1 will display live footage back to the operator to guide them through immersive underwater exploration. Archimedes-1 is designed as a sub you can quickly deploy in order to investigate sealife or to examine underwater structures quickly to find maintinance issues.
